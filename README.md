# 📊 Adventure Works Power BI Dashboard

## 📖 Overview
This project is a **4-page Power BI dashboard** for **AdventureWorks**, a global retail company.

The report follows a **top-down analytical flow**, starting with a high-level business overview and drilling down into:

- Regional sales performance
- Product performance
- Customer behavior insights

![Overview image of Adventure Works report](https://github.com/DragonTP/AdventureWorks/blob/affbd3412b956d3fdf3359e775366eb70da3aa52/adventurework%20project%20image.jpg)

---

## 🛠️ Tech Stack

**Power BI**

Used for the entire workflow:

- Data cleaning  
- Data modeling  
- Dashboard visualization  

---

## ✨ Key Features

### 📊 Page 1: Executive Summary

Tracks the **overall business performance** with key KPIs:

- **Total Revenue**
- **Profit**
- **Orders**
- **Return Rate**

Visualizations include:

- Revenue **trend over time**
- **Top-selling product categories**

---

### 🌍 Page 2: Regional Analysis

Provides a **global view of sales performance**.

Features include:

- Sales distribution across:
  - **Europe**
  - **North America**
  - **Pacific**

- **Interactive map visualization**
- Order volume by **country and city**
- Region selection that **filters the dashboard**

---

### 📦 Page 3: Product Detail

Focuses on **product-level performance analysis**.

Features include:

- Drill-down into **specific products**
- Monthly performance tracking

Visualizations include:

- **Gauge charts** comparing:
  - Revenue vs Target
  - Profit vs Target
  - Orders vs Target

- **Price Adjustment Slider (What-if parameter)**  
  Simulates how **price changes impact profit margins**.

---

### 👥 Page 4: Customer Analysis

Analyzes the company’s **17.4K unique customers**.

Key insights include:

- Customer segmentation by:
  - **Occupation**
  - **Income level**

- Identification of **high-value customers**
